Discuss the scope of var,let and const

+

Var:
Var is a funtional scope. It means it can be accessed from outside of an function.
Let:
Let is a block scope. It means it can be accessed only within the block where it is defined.
Const:
Const is a block scope and also has a value that cannot be changed.
